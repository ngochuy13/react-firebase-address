# HuyPham-boilerplate

### Install
```sh
git clone https://github.com/ngochuy13/react-firebase-address
cd frontend.thousandhands.com.git
npm install
```

### Run the app from a dev server
```sh
npm run start
```

### Build the app
```sh
npm run build
```

### Host the app from a server (app must have been built)
```sh
npm run serve
```
